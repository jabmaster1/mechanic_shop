# Mechanic Shop

A simple Flask-based application to manage customers, service tickets, and mechanics for an auto repair shop. The application uses **Flask**, **SQLAlchemy**, and **Marshmallow** for ORM and serialization.

---

## Features

- Manage customers (CRUD)
- Manage mechanics (CRUD)
- Create and track service tickets
- Assign/remove mechanics to/from service tickets
- Retrieve all service tickets
- Added Inventory management and ability to assign inventory to tickets
